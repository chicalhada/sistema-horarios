# Instalação

## Requisitos

Para executar a aplicação é necessário possuir:

- PHP 8 ou superior
- SQLite
- Apache (XAMPP)

---

## Instalação

### 1.

Copiar a pasta **sistema-horarios** para:

```
C:\xampp\htdocs\
```

---

### 2.

Abrir o XAMPP.

Iniciar o serviço Apache.

---

### 3.

Abrir o navegador.

```
http://localhost/sistema-horarios/public
```

---

## Base de Dados

A aplicação utiliza SQLite.

Caso a base de dados ainda não exista será criada automaticamente.

---

## Estrutura

A pasta **data** contém os dados da aplicação.

A pasta **public** contém as páginas acessíveis pelo navegador.

A pasta **src** contém a lógica da aplicação.

A pasta **views** contém as páginas apresentadas ao utilizador.
