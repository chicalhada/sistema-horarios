# Guia do Utilizador

## Login

O administrador deverá iniciar sessão através da página de login.

Após a autenticação será possível importar novos horários.

---

## Importação de Horários

1. Entrar na área de administração.
2. Abrir a página de Upload.
3. Selecionar um ficheiro CSV.
4. Clicar em Importar.

Os horários serão automaticamente guardados na base de dados.

---

## Consultar Salas

Aceder à lista de salas.

Selecionar a sala pretendida.

O sistema apresenta automaticamente o respetivo horário.

---

## Gerar QR Code

Na página da sala existe um botão para gerar o QR Code.

O QR Code permite abrir diretamente o horário da sala utilizando um telemóvel.

---

## Recomendações

Utilizar apenas ficheiros CSV válidos.

Não alterar o formato do ficheiro.

Verificar sempre se a importação terminou com sucesso.
