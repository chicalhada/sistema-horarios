# Resolução de Problemas

## Não consigo iniciar sessão

Verifique:

- Nome de utilizador.
- Palavra-passe.

---

## O CSV não importa

Possíveis causas:

- ficheiro vazio;
- formato inválido;
- colunas em falta.

Verifique o ficheiro antes de importar.

---

## Não aparecem salas

Confirme que o CSV foi importado corretamente.

---

## Erro na Base de Dados

Verifique se a pasta **data** possui permissões de escrita.

---

## QR Code não aparece

Verifique se existe a pasta:

```
public/qr-codes
```

---

## Como reportar um erro

Sempre que encontrar um problema indique:

- descrição;
- passos para reproduzir;
- navegador;
- sistema operativo;
- captura de ecrã.
