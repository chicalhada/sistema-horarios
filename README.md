# Sistema de Horários

## Sobre o Projeto

O Sistema de Horários é uma aplicação web desenvolvida em PHP que permite gerir e consultar os horários das salas de uma instituição de ensino.

O sistema foi criado para facilitar a organização dos horários, permitindo a importação de dados através de ficheiros CSV e a consulta rápida da ocupação das salas.

Além disso, disponibiliza QR Codes para que qualquer utilizador possa consultar um horário diretamente através do telemóvel.

---

## Funcionalidades

O sistema possui as seguintes funcionalidades:

- Área de administração
- Login de administrador
- Importação de horários através de ficheiros CSV
- Armazenamento em base de dados SQLite
- Consulta dos horários por sala
- Listagem de todas as salas
- Geração de QR Codes
- Interface Web responsiva
